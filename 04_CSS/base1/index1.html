Форматы файлов.

.DOC. или doc (от англ. document)-расширение имени файла, используемое для файлов, предоставляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файлов без формирования, однако позже стало использоваться для двоичных форматов с разметкой. Document MS World
Portable Document Format(PDF )-межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использоваем ряда возможностей языка Postscript. Документ PDF
ZIP-попоулярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет расширение zip и хранит в сжатом или несжатом виде один или несколько файлов, которое можно из него извлечь путем распаковки с помощью специальной программы. Архив
